# Tests / Screenshots

Screenshots capturados durante testes automatizados.

## Propósito

Esta pasta armazena screenshots de:

- Testes de UI
- Testes de acessibilidade
- Testes de responsividade
- Comparações visuais (visual regression)
- Evidências de bugs

## Uso

Screenshots são gerados automaticamente durante a execução de testes específicos e podem ser usados para:

- Documentação visual
- Debugging
- Comparação de mudanças visuais
- Relatórios de testes

## Convenções

- Nomes descritivos: `test-name-component-state.png`
- Organize por feature ou componente
- Não commite screenshots desnecessários
- Use .gitignore para screenshots temporários
