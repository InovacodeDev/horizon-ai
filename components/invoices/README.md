# Components / Invoices

Componentes relacionados ao gerenciamento e visualização de notas fiscais (NFe).

## Arquivos

- **InvoiceCard.tsx** - Card para exibir resumo de uma nota fiscal
- **InvoiceListSkeleton.tsx** - Skeleton loader para lista de notas fiscais
- **PriceComparisonTable.tsx** - Tabela comparativa de preços de produtos entre diferentes NFes
- **PriceHistoryChart.tsx** - Gráfico de linha mostrando histórico de preços de um produto
- **ResponsiveChartContainer.tsx** - Container responsivo para gráficos (Recharts)
- **ShoppingListBuilder.tsx** - Construtor de lista de compras baseado em produtos de NFes

## Funcionalidades

- Visualização de NFes com detalhes de produtos
- Comparação de preços entre estabelecimentos
- Tracking de histórico de preços
- Geração de listas de compras inteligentes
