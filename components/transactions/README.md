# Components / Transactions

Componentes específicos para gerenciamento de transações.

## Arquivos

- **ImportTransactionsModal.tsx** - Modal completo para importação de transações (OFX, CSV, PDF)
- **ImportPreview.tsx** - Preview das transações antes de confirmar importação
- **ImportHistory.tsx** - Histórico de importações realizadas
- **index.ts** - Exports centralizados

## Funcionalidades

- Upload de arquivos OFX, CSV e PDF
- Parsing automático de transações
- Preview com possibilidade de edição antes de importar
- Mapeamento de categorias
- Histórico de importações com status
