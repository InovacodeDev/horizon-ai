## 🎓 Como Usar o Projeto

### 1. Setup Inicial

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Run development server
npm run dev
```

### 2. Storybook

```bash
# Start Storybook
npm run storybook

# Build Storybook
npm run build-storybook

# Deploy Storybook
# npm run deploy-storybook
```

### 3. Performance Analysis

```bash

# Run performance tests
npm run test:perf

# Run Lighthouse CI
```

```bash
# Run all tests

npm run test:coverage

# Run accessibility tests
npm run test:a11y

npm run test:perf
```
